◆ CharToDigit() [2/2]

uint base )
inlinestatic

this method changes a character 'c' into its value (if there can't be a correct value it returns -1)

for example:

  • c=2, base=10 -> function returns 2
  • c=A, base=10 -> function returns -1
  • c=A, base=16 -> function returns 10
