\doxysubsection{Detailed Description}
A small class used for reporting \textquotesingle{}reference\textquotesingle{} errors

In the library is used macro TTMATH\+\_\+\+REFERENCE\+\_\+\+ASSERT which can throw an exception of this type

from version 0.\+9.\+2 this macro is removed from all methods in public interface so you don\textquotesingle{}t have to worry about it \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}

If you compile with gcc you can get a small benefit from using method Where() (it returns std\+::string) with the name and the line of a file where the macro TTMATH\+\_\+\+REFERENCE\+\_\+\+ASSERT was used)
