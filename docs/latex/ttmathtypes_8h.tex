\doxysubsection{Detailed Description}
constants used in the library 

As our library is written in header files (templates) we cannot use constants like \textquotesingle{}const int\textquotesingle{} etc. because we should have some source files .cpp to define this variables. Only what we can have are constants defined by \#define preprocessor macros.

All macros are preceded by TTMATH\+\_\+ prefix
