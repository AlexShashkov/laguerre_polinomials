\doxysubsection{Detailed Description}
a small class used for reporting errors

in the library is used macro TTMATH\+\_\+\+ASSERT which (if the condition in it is false) throw an exception of this type

if you compile with gcc you can get a small benefit from using method Where() (it returns std\+::string) with the name and the line of a file where the macro TTMATH\+\_\+\+ASSERT was used)
