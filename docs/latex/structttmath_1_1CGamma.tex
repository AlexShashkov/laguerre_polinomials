this method prepares some coefficients\+: factorials and Bernoulli numbers stored in \textquotesingle{}fact\textquotesingle{} and \textquotesingle{}bern\textquotesingle{} objects

how many values should be depends on the size of the mantissa -\/ if the mantissa is larger then we must calculate more values for a mantissa which consists of 256 bits (8 words on a 32bit platform) we have to calculate about 30 values (the size of fact and bern will be 30), and for a 2048 bits mantissa we have to calculate 306 coefficients

you don\textquotesingle{}t have to call this method, these coefficients will be automatically calculated when they are needed

you must note that calculating these coefficients is a little time-\/consuming operation, (especially when the mantissa is large) and first call to \doxylink{namespacettmath_aaf1312e301f4800c67164f4eb8a5e017}{Gamma()} or \doxylink{namespacettmath_ae0a248c8618c9ac4e3edf6095fe31f92}{Factorial()} can take more time than next calls, and in the end this is the point when \doxylink{structttmath_1_1CGamma_a0741221115113a71d3c9b183a46042d6}{Init\+All()} comes in handy\+: you can call this method somewhere at the beginning of your program
